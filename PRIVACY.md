# AuraGrowthStudio — Privacy Policy

Last updated: 2026-05-09

This Pinterest application is an internal automation tool for AuraGrowthStudio, a wedding stationery shop on Etsy (https://etsy.com/shop/AuraGrowthStudio). It is operated by a single user (the shop founder) for the purpose of scheduling Pinterest pins.

## Data collection and use

- The application accesses the founder's own Pinterest account via OAuth.
- It does **NOT** collect, share, or sell any third-party user data.
- API tokens are stored locally on the founder's machine.
- No analytics, tracking, or advertising data is collected.
- No data is shared with any third party beyond Pinterest's own systems.
